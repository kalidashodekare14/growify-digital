A modern and responsive digital agency website built with React, Vite, and Tailwind CSS. This project was developed as part of a technical assessment for a React Developer Internship.

##  Features

- Fully responsive design
- Clean and modern UI
- Reusable component-based architecture
- Multiple pages (Home, Services, About, Blog, Contact)
- Smooth navigation using React Router
- Interactive UI elements and hover effects
- Contact form and map integration
